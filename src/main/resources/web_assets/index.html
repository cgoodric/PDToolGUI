PROMOTION & DEPLOYMENT TOOL

PDTool is a field-developed solution for automating the process of deploying CIS resources from development to test to production. It assists with deploying resources (either through CAR files or through a version control system), managing users and groups, updating permissions, resetting data source connection information and login credentials, relocating or rebinding resources, etc.
